# Test Unitarios para comprobar que la implementación de OpenGIS Coordinate Transformations de Digi3D.NET supera el 100 de las pruebas

Este proyecto de test unitarios crea una fábrica de sistemas de coordenadas denominada GIGS y un proyecto de test unitarios para comprobar
que [Digi3D.NET](http://www.digi21.net/digi3d) supera todas las pruebas de [Geospatial integrity of geoscience software (GIGS)](http://www.epsg.org/gigs.html)


## Paquetes requeridos

* Digi21.OpenGIS (incluido con una instalación de Digi3D.NET)
